Mock Foods ID

Mock Foods ID containing food items from Indonesia for your application prototype.

View Docs View Repos

Description

This API provides a collection of delicious Indonesian foods, including satay (sate), rendang, nasi goreng, and more. Use this API to develop your culinary website or restaurant application and showcase the rich flavors of Indonesian cuisine.

Example Using Axios

Get Foods

axios.get('http://localhost:3002/foods')

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Get Foods Using Limit & Sort

axios.get('http://localhost:3002/foods?limit=12&sort=desc')

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Get Categories

axios.get('http://localhost:3002/foods/categories')

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Get Foods Category

axios.get('http://localhost:3002/foods/category/entree')

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Get Food by ID

axios.get('http://localhost:3002/foods/1')

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Post Food

axios.post('http://localhost:3002/foods', {

"name": "Nasi Goreng",

"category": "Main Dish",

"description": "A popular Indonesian fried rice with a mix of ingredients and spices.",

"price": 10000,

"image": "nasi_goreng.gif"

})

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Edit Food

axios.put('http://localhost:3002/foods/12', {

"name": "Nasi Goreng",

"category": "Main Dish",

"description": "A popular Indonesian fried rice with a mix of ingredients and spices.",

"price": 10000,

"image": "nasi_goreng.gif"

})

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Delete Food

axios.delete('http://localhost:3002/foods/1')

.then((response) => { console.log(response.data); })

.catch((error) => { console.error(error); });

Note: Please be aware that all POST, PUT, and DELETE routes do not add, update or delete food data from the database. Instead, they are designed to return a success message if the request is made properly. The food data remains intact in the database for reference and can be retrieved as needed.
